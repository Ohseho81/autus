# AUTUS UI/UX Design Guide v1.0

**대상**: UI/UX 디자이너  
**목적**: AUTUS의 디자인 원칙과 제약조건

---

## 1. 핵심 원칙

### "Show, don't tell"

```
사용자에게:
├── 물리를 보여준다     ✅
├── 숫자를 보여준다     ✅
├── 파형을 보여준다     ✅
├── 판단을 말하지 않는다 ❌
├── 조언을 말하지 않는다 ❌
└── 추천을 말하지 않는다 ❌
```

### "One screen. No more pages."

```
├── 메인 화면 1개 (항상 보임)
├── 오버레이 3개 (스와이프로 전환)
└── 페이지 이동 없음
```

## 2. 색상 체계

### Primary Colors

| 용도 | 색상 | Hex | 의미 |
|------|------|-----|------|
| **양(+) 방향** | Teal | `#00d4aa` | 증가/흐름 |
| **음(-) 방향** | Red | `#ff3b5c` | 감소/위험 |
| **보조** | Blue | `#00b4d8` | Flow/중립 |
| **배경** | Black | `#000000` | 깊이감 |
| **밝은 강조** | Bright Teal | `#00ffcc` | 하이라이트 |

### ⚠️ 중요

```
색상은 "좋음/나쁨"이 아니라 "방향"만 표시합니다.

Teal = 증가 방향 (좋다 ❌)
Red = 감소 방향 (나쁘다 ❌)
```

## 6. 홀로그램 효과

### 구현 요소

| 효과 | 값 | 용도 |
|------|-----|------|
| **Transparency** | 60-80% | 깊이감 |
| **Blur** | 20-40px | 배경 분리 |
| **Glow** | Teal, 30-60px | 빛 산란 |
| **Particles** | 30개 | 양자 느낌 |
| **Reflection** | 5% opacity | 바닥 반사 |

### CSS 예시

```css
/* 홀로그램 요소 */
.holo-element {
  background: rgba(0, 212, 170, 0.1);
  border: 1px solid rgba(0, 212, 170, 0.3);
  backdrop-filter: blur(20px);
  box-shadow: 
    0 0 30px rgba(0, 212, 170, 0.2),
    inset 0 0 20px rgba(0, 212, 170, 0.1);
}

/* 글로우 효과 */
.glow {
  box-shadow: 0 0 40px var(--teal);
  text-shadow: 0 0 20px var(--teal);
}
```

|------|------|
