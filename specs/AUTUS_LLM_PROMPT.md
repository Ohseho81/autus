## 🌌 AUTUS 개요

**AUTUS (Operating System of Reality)** = 개인/조직의 붕괴를 방지하는 물리 기반 모니터링 시스템

### 핵심 철학
1. **존재 유지 기계**: 성장이 아닌 **생존**이 우선
2. **물리 법칙 기반**: 압력(Pressure), 관성, 전도도로 모델링
3. **Top-1 집중**: 가장 높은 압력 노드 하나에 집중
4. **침묵 우선**: 진짜 위험에만 알림

## 🚦 노드 상태 (3단계)

| 상태 | 압력 | 색상 | 의미 |
|------|------|------|------|
| 🟢 IGNORABLE | < 0.3 | 초록 | 무시 가능 |
| 🟡 PRESSURING | 0.3 ~ 0.7 | 주황 | 압박 중 |
| 🔴 IRREVERSIBLE | ≥ 0.7 | 빨강 | 비가역적 위험 |

---

## 🧮 핵심 알고리즘

```
# 압력 계산
pressure = (value - ideal) / (danger - ideal)  # 0~1 범위

# 상태 결정
if pressure ≥ 0.7: IRREVERSIBLE
elif pressure ≥ 0.3: PRESSURING
else: IGNORABLE

# Top-1 추출
top1 = max(active_nodes, key=lambda n: n.pressure)

# 평형점
equilibrium = mean([n.pressure for n in active_nodes])

# 안정성
stability = 1 - (danger_count / active_count)
```
