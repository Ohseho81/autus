```

```

**질문:**
1. **1차 타겟 사용자는 누구인가요?**
   - [ ] 개인 사업자 (1인 기업, 프리랜서)
   - [ ] 중소기업 대표/임원
   - [ ] 영업/세일즈 직군
   - [ ] 네트워킹이 많은 전문직 (변호사, 컨설턴트)
   - [ ] 일반 직장인
   - [ ] 학원/교육 사업자 (기존 education 팩 관련)

2. **예상 노드(인맥) 규모는?**
   - 평균 사용자: 50명? 150명? 500명?
   - 파워 유저: 1,000명? 5,000명?

3. **연령대는?**
   - 30~40대 사업가 중심?

### **[11] 법적/규제 이슈 - 반드시 검토 필요**

```
현재: 검토 안됨 (리스크)
```

**질문:**
1. **개인정보보호법 (PIPA) 대응은?**
   - 노드(타인) 정보 저장 시 동의 필요 여부
   - 제3자 정보 수집의 법적 한계
   - 정보주체 열람/삭제 요청 대응

2. **자동 메시지 발송의 법적 이슈는?**
   - 카카오톡: 비즈니스 채널 필수 (개인 계정 자동화 불가)
   - 이메일: CAN-SPAM Act 준수 (수신거부 링크 필수)
   - SMS: 사전 동의 필수

3. **"자동 차단"의 법적 리스크는?**
   - 특정인 차별로 해석될 가능성
   - 명예훼손 소지 (시너지 점수 공개 시)

4. **이용약관/개인정보처리방침 준비 시점은?**
   - MVP 전? MVP 후?
