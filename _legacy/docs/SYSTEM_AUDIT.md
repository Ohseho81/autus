### **[1] 마스터 키 시스템 - 주권 구조**

```
현재 설계: "불가역적 백도어", "쉐도우 키" 구조
```

**질문:**
1. **인증 체계**: 마스터 키는 어떤 형태인가요?
   - [ ] 하드웨어 키 (YubiKey, Ledger 등)
   - [ ] 소프트웨어 토큰 (JWT + 비대칭 암호화)
   - [ ] 생체 인증 연동
   - [ ] 복합 (MFA + 지리적 제한)

2. **"주권자만 접근"의 범위**: 어떤 기능까지 마스터 키로 제어하나요?
   - 모든 유저 데이터 열람?
   - 클러스터 강제 재배치?
   - 자동화 규칙 오버라이드?
   - 수익 분배 조정?
