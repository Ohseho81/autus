## Phase 2: 발아 (Month 7-12)

### 목표
```
스포츠 교육 500개 (농구 + 태권도 + 수영 + 축구)
```

### 전략: B2B2C 바이럴 시스템

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   🔄 바이럴 루프                                                │
│                                                                 │
│   학부모 A (농구 학원)                                          │
│        │                                                        │
│        ├── "이 앱 너무 편해요"                                  │
│        │                                                        │
│        ▼                                                        │
│   아이 태권도 학원에 추천                                        │
│        │                                                        │
│        ├── "학부모님이 추천하셔서요"                            │
│        │                                                        │
│        ▼                                                        │
│   태권도 관장 가입                                               │
│        │                                                        │
│        ├── 학부모 A에게 1개월 무료                              │
│        │                                                        │
│        ▼                                                        │
│   태권도 학부모들도 다른 학원에 추천                             │
│        │                                                        │
│        └── 무한 반복                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

| 학원장 | 학원장 | 2개월 무료 |

## 📊 핵심 실행 원칙

### 1. 속도 > 완벽

```
완벽한 제품 만들고 런칭 ❌
MVP로 빨리 시장 검증 ✅

"Done is better than perfect"
```

### 2. 무료로 중독시키기

```
처음부터 유료 ❌
무료로 습관 → 유료 전환 ✅

"Free until they can't live without it"
```

### 3. 고객이 영업한다

```
영업사원 채용 ❌
학부모가 추천 ✅

"Best marketing is word of mouth"
```

### 4. 사람 대신 시스템

```
고객 늘면 직원 늘림 ❌
시스템이 자동 처리 ✅

"Scale with code, not people"
```

### 5. 한 놈만 패기

```
모든 시장 동시 공략 ❌
농구 독점 → 확장 ✅

"Dominate one, then expand"
```
