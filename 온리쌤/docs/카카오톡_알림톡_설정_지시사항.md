## ✅ 작업 체크리스트

### 1단계: 카카오 개발자 콘솔 (30분)

**웹사이트**: https://developers.kakao.com

- [ ] 카카오 계정으로 로그인 (사업자로 가입)
- [ ] [내 애플리케이션] → [애플리케이션 추가하기] 클릭
- [ ] 앱 이름: **"ATB"** 입력
- [ ] 사업자명: **"올댓바스켓"** 입력
- [ ] [저장] 클릭
- [ ] 생성된 앱 클릭 → [앱 키] 메뉴
- [ ] **REST API 키** 복사하여 아래에 기록:
  ```
  REST API 키: _________________________________
  ```

---

### 2단계: 카카오톡 채널 개설 (30분)

**웹사이트**: https://center-pf.kakao.com

- [ ] 카카오 계정으로 로그인
- [ ] [새 채널 만들기] 클릭
- [ ] 채널명: **"ATB"** 또는 **"올리쌤"** 입력
- [ ] 검색용 ID: **@atb** (사용 가능한 ID로)
- [ ] 프로필 이미지 업로드 (ATB 로고)
- [ ] [저장] 후 채널 개설 완료
- [ ] 채널 관리자 화면에서 **Sender Key** 확인 및 기록:
  ```
  Sender Key: @_________________________________
  ```

---

### 3단계: 카카오톡 비즈니스 계정 (30분)

**웹사이트**: https://business.kakao.com

- [ ] 카카오 계정으로 로그인
- [ ] [새 비즈니스 계정 만들기] 클릭
- [ ] 사업자 정보 입력
  - 사업자등록번호
  - 대표자명
  - 사업장 주소
- [ ] [비즈메시지] → [발신 프로필 관리] 클릭
- [ ] [새 프로필 등록] 클릭
- [ ] 프로필명: **"ATB"**
- [ ] 위에서 만든 카카오톡 채널 연결
- [ ] [심사 신청] 클릭 (승인까지 1~2영업일 소요)
- [ ] ⚠️ **심사 승인 메일 받으면 바로 알려주세요**

---

### 4단계: Solapi 가입 및 충전 (20분)

**웹사이트**: https://solapi.com

- [ ] [회원가입] 클릭 (이메일 또는 카카오 계정)
- [ ] [콘솔] → [API 키 관리] 메뉴
- [ ] [새 API 키 생성] 클릭
- [ ] API Key 복사하여 기록:
  ```
  API Key: _________________________________
  ```
- [ ] API Secret 복사하여 기록:
  ```
  API Secret: _________________________________
  ```
- [ ] [결제] → [포인트 충전] 클릭
- [ ] **10,000원 이상 충전** (카드 결제)
- [ ] 충전 완료 확인

---

### 5단계: 알림톡 템플릿 등록 (1시간)

**웹사이트**: Solapi 콘솔 → [알림톡] → [템플릿 관리]

아래 **5개 템플릿**을 각각 등록하세요:

#### 템플릿 1: 출석 확인
- [ ] 템플릿 이름: **출석 확인**
- [ ] 템플릿 코드: **ATB_ATTENDANCE**
- [ ] 내용:
  ```
  안녕하세요, #{student_name} 학부모님!

  오늘 #{date} #{time} 수업에 출석했습니다. ✅

  #{coach_name} 코치님과 함께한 #{duration}분 수업이었습니다.

  좋은 하루 되세요!
  - ATB 팀
  ```
- [ ] [저장] → [심사 요청] 클릭

#### 템플릿 2: 수업 리마인더
- [ ] 템플릿 이름: **수업 리마인더**
- [ ] 템플릿 코드: **ATB_LESSON_REMIND**
- [ ] 내용:
  ```
  #{student_name} 학부모님,

  1시간 후 #{time}에 #{lesson_name} 수업이 있습니다! 🏀

  📍 장소: #{location}
  👨‍🏫 코치: #{coach_name}

  잊지 마세요!
  - ATB 팀
  ```
- [ ] [저장] → [심사 요청] 클릭

#### 템플릿 3: 결제 독촉
- [ ] 템플릿 이름: **결제 독촉**
- [ ] 템플릿 코드: **ATB_PAYMENT_DUE**
- [ ] 내용:
  ```
  #{student_name} 학부모님,

  #{month}월 수업료 #{amount}원이 미납 상태입니다.

  💳 납부 기한: #{due_date}까지
  📱 결제하기: #{payment_link}

  문의사항이 있으시면 언제든 연락주세요.
  - ATB 팀
  ```
- [ ] [저장] → [심사 요청] 클릭

#### 템플릿 4: 피드백 요청
- [ ] 템플릿 이름: **피드백 요청**
- [ ] 템플릿 코드: **ATB_FEEDBACK**
- [ ] 내용:
  ```
  #{student_name} 학부모님,

  오늘 수업은 어떠셨나요? 💬

  #{coach_name} 코치님의 피드백:
  "#{feedback_message}"

  다음 목표: #{next_goal}

  응원합니다! 🎉
  - ATB 팀
  ```
- [ ] [저장] → [심사 요청] 클릭

#### 템플릿 5: 환영 메시지
- [ ] 템플릿 이름: **환영 메시지**
- [ ] 템플릿 코드: **ATB_WELCOME**
- [ ] 내용:
  ```
  #{student_name} 학부모님, 환영합니다! 🎊

  ATB 가족이 되신 것을 진심으로 축하드립니다.

  📱 앱 다운로드: #{app_link}
  👨‍🏫 담당 코치: #{coach_name}
  📅 첫 수업: #{first_lesson_date}

  함께 성장해요!
  - ATB 팀
  ```
- [ ] [저장] → [심사 요청] 클릭
