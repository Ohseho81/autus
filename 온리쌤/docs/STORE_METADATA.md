## 앱 설명 (한국어)

### 짧은 설명 (Google Play - 80자 이내)

### 전체 설명

온리쌤은 체육학원 코치와 관리자를 위한 스마트 학원 운영 앱입니다.

▶ 코치를 위한 초간단 수업 관리
• 버튼 3개로 수업 시작부터 종료까지
• QR 코드 자동 출석 체크
• 수업 중 영상 촬영 → 학부모 자동 공유

▶ 관리자를 위한 통합 대시보드
• 학생 등록, 수강 현황 한눈에 관리
• 출석률, 수업 현황 실시간 확인
• 결제 및 미납 관리

▶ 학부모 연결
• 카카오 알림톡으로 출석·결석 자동 알림
• 수업 영상 자동 공유
• 보충수업 자동 안내

주요 기능:
✓ QR 코드 출석 체크
✓ 수업 스케줄 관리
✓ 수업 영상 촬영 및 공유
✓ 학생·학부모 관리
✓ 결제 및 수강 관리
✓ 카카오 알림톡 자동 발송
