## 📝 게시글 1: 가치 제공 글 (1주차)

### 제목
**[노하우] 퇴원 위험 학생, 미리 알 수 있는 5가지 신호**

### 본문

```
안녕하세요, 원장님들.

10년차 학원 운영하면서 느낀 건데요,
퇴원생이 생기기 전에 항상 "신호"가 있더라고요.

제가 경험한 퇴원 위험 신호 5가지 공유드립니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚨 1. 출석률 급락 (가장 확실한 신호)
- 평소 100% → 갑자기 80% 미만
- 특히 "아파서", "학교 일정" 핑계가 늘면 위험

🚨 2. 숙제 미제출 증가
- 평소 잘 하다가 2주 연속 미제출
- "바빠서"라는 말이 나오면 관심 필요

🚨 3. 학부모 연락 두절
- 카톡 읽씹, 전화 안 받음
- 상담 요청해도 "바쁘다" 회피
- 이미 다른 학원 알아보는 중일 수 있음

🚨 4. 시험 후 성적 하락
- 중간/기말 끝나고 점수 떨어지면
- 학부모가 "효과 없다" 생각하기 시작

🚨 5. 수업 태도 변화
- 강사가 "요즘 집중 안 해요" 피드백
- 친구들과 어울리지 않음

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 대응 방법

위 신호 중 2개 이상 보이면 바로 상담하세요.

저는 매주 월요일 아침에 이런 학생들 체크하고,
화요일까지 학부모 연락합니다.

"요즘 아이 학습 상황 어떠세요?" 
이렇게 먼저 물어보면 대부분 마음 열어요.

퇴원 막는 건 새 학생 모집보다 쉽습니다.
기존 학생 한 명 = 신규 학생 3명 모집 효과

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

원장님들은 어떤 신호로 퇴원 위험 감지하시나요?
댓글로 노하우 공유 부탁드려요! 🙏
```

### 제목
**[무료 테스트] 퇴원 위험 학생 미리 찾아주는 AI 시스템, 원장님 3분 모십니다**

### 본문

```
안녕하세요, 원장님들.

저번에 퇴원 위험 신호 글 올렸던 사람입니다.
(반응 좋아서 감사했습니다 🙏)

그때 댓글 보면서 느낀 건데,
많은 원장님들이 "신호는 아는데 일일이 체크할 시간이 없다"고 하시더라고요.

그래서 저희 팀이 만들고 있는 시스템 소개드립니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 AUTUS - 퇴원 위험 자동 감지 시스템

[기능]
✅ 출석률, 숙제 제출, 성적 변화 자동 분석
✅ 퇴원 위험 학생 실시간 알림
✅ 학부모 상담 스크립트 AI 자동 생성
✅ 상담 결과 기록 및 추적

[작동 방식]
1. 학원 데이터 연동 (학원나라, 엑셀 등)
2. AI가 매일 분석
3. "이 학생 다음 달 퇴원 확률 80%" 알림
4. 상담 스크립트까지 자동 생성

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎁 파일럿 프로그램 (무료)

[대상]
- 학생 50~200명 규모 학원
- 퇴원 관리에 어려움 느끼시는 원장님
- 새로운 시스템 테스트에 관심 있으신 분

[조건]
- 기간: 3개월 완전 무료
- 주 1회 피드백 미팅 (30분, 화상)
- 퇴원/미수금 데이터 공유

[과금 방식] ← 이게 핵심입니다
- 3개월 후, 실제로 퇴원 방지된 건수만 과금
- 효과 없으면 0원
- 효과 있으면 퇴원 방지 가치의 5~10%

예시)
- 퇴원 방지 5건 × 월 35만원 × 6개월 × 10%
- = 105만원 (방지 안 했으면 1,050만원 손실)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

간단히 학원 상황 여쭤보고,
저희 시스템이 맞는지 10분 통화로 확인드리겠습니다.

선착순 3~5곳만 모집합니다.
(제대로 피드백 드리려면 많이 못 받아요)

질문 있으시면 댓글로 남겨주세요!
```
