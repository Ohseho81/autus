## 0단계 — **우주 (Universal Physics)**

### "판단 이전의 법칙"

**정의**
- AUTUS의 최상위 헌법
- 인간 판단·정책·의견이 개입할 수 없는 영역

**핵심 물리**
| 기호 | 명칭 | 정의 |
|------|------|------|
| ψ | 비가역성 | 되돌림 불가능 정도 |
| ΔṠ | 엔트로피 가속 | 무질서 증가 속도 |
| UC | 책임 수용 한계 | 개인이 감당 가능한 최대 책임 |
| E | 에너지 예산 | 가용 자원 총량 |
| θ | 종결 임계 | 자동 잠금 발동 기준 |

**Gate 규칙**
```
G1: ΔṠ > θ   → 엔트로피 임계 초과
G2: Load > UC → 책임 한계 초과
G3: E < 0     → 에너지 고갈

→ 자동 LOCK → Afterimage
```

**결과**
- 자동 종결
- 되돌림 불가
- 설명 없음

## 1단계 — **은하 (Galaxy of Work / Work Genome)**

### "업무는 설계되지 않는다, 선별된다"

**정의**
- 모든 업무를 생명체처럼 취급
- 증식·소멸·변이 규칙만 존재

**구성**
- Work Genome Tree
- 업무 물성(DNA): 질량, 비가역성, 실패 비용
- 자연 선택 루프

**7대 물리 법칙**
| 법칙 | 기호 | 특성 |
|------|------|------|
| 중력 | G | 다른 업무를 끌어당김 |
| 관성 | I | 시작/정지 어려움 |
| 엔트로피 | Ω | 방치 시 무질서 증가 |
| 운동량 | p | 멈추면 충격 |
| 파동 | λ | 주기적 반복 |
| 마찰 | μ | 진행 저항 |
| 공명 | ω₀ | 동조 시 증폭 |

**결과**
- 570개 업무는 템플릿이 아니라 **종(種)**
- 인간은 창조자가 아니라 **환경**

**구성**
- 모든 노드에 `lat/lng`
- 행정·규제 경계
- 거리·밀집·경계 기반 전파 커널

**결과**
- 인과 = 공간 전파
- 좌표 없는 결정은 존재 불가
