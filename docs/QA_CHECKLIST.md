## 1️⃣ SOLAR — 기능 테스트

### 렌더링
- [ ] 페이지 로드 시 SOLAR 즉시 표시
- [ ] 태양 중앙 정렬
- [ ] 궤도 객체 3개 표시 (모바일 2개)
- [ ] 궤도 회전 부드러움 (끊김 없음)

### 상태 변화
- [ ] RISK 0% → 녹색 상태
- [ ] RISK 40% → 노란색 상태
- [ ] RISK 70% → 빨간색 상태
- [ ] RISK 80%+ → 임계 상태 (회전 정지)

### 색상
- [ ] GREEN: `#6EDC9A` 정확
- [ ] YELLOW: `#FFD166` 정확
- [ ] RED: `#EF476F` 정확

## 2️⃣ RISK % — 기능 테스트

### 표시
- [ ] 숫자만 표시 (단위 `%` 포함)
- [ ] 폰트 크기 72px (모바일 56px)
- [ ] 위치 고정 (스크롤 무관)

### 상태 연동
- [ ] 색상 즉시 변경 (transition 없음)
- [ ] API 응답과 동기화

---

## 3️⃣ ACTION 버튼 — 기능 테스트

### 출현 조건
- [ ] RISK < 40%: 버튼 숨김
- [ ] RISK 40-69%: 3초 후 "확인" 버튼
- [ ] RISK 70%+: 즉시 "조치" 버튼

### 인터랙션
- [ ] 클릭 1회만 허용
- [ ] 더블클릭 방지
- [ ] 클릭 후 즉시 비활성화
- [ ] 클릭 후 AUDIT 전환

### 스타일
- [ ] 모바일: 64px 원형
- [ ] 터치 영역 충분
- [ ] 엄지 도달 가능

---

## 4️⃣ AUDIT — 기능 테스트

### 토큰 낙하
- [ ] 시작 위치: SOLAR 중심
- [ ] 낙하 시간: 정확히 0.9초
- [ ] 가속 곡선: Ease-in (느림 → 빠름)
- [ ] Bounce 없음

### 충돌
- [ ] 화면 진동: 2px / 60ms
- [ ] 사운드 재생 (데스크톱)
- [ ] 햅틱 피드백 (모바일)
- [ ] 모든 애니메이션 정지

### 완료 상태
- [ ] "SEALED" 텍스트 페이드인
- [ ] 추가 반응 없음
- [ ] 오버레이 클릭 시 닫힘
