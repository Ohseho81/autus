## 1. 시작하기

### 1.1 접속하기

웹 브라우저에서 다음 주소로 접속합니다:

```
https://autus.app (프로덕션)
http://localhost:8000 (로컬 개발)
```

### 1.2 첫 화면

접속하면 **Physics Map**이 표시됩니다. 이 맵은 글로벌 자금/무역/인적 흐름을 시각화합니다.

### 1.3 키보드 단축키

| 키 | 기능 |
|----|------|
| `1` | 미국 지역 이동 |
| `2` | 영국 지역 이동 |
| `3` | 유럽 지역 이동 |
| `4` | 중국 지역 이동 |
| `5` | 일본 지역 이동 |
| `6` | 인도 지역 이동 |
| `7` | 중동 지역 이동 |
| `0` / `W` | 전체 세계 보기 |
| `M` | 모션(흐름선) 표시/숨김 |
| `L` | 라벨 표시/숨김 |
| `S` / `/` | 검색창 열기 |
| `ESC` | 패널 닫기 |

## 2. Physics Map 사용법

### 2.1 노드 (Node)

맵에 표시되는 원형 마커가 **노드**입니다.

#### 노드 계층 (5-Tier System)

| Tier | 색상 | 설명 | 예시 |
|------|------|------|------|
| **T1** | 🟡 금색 | 허브 | 미국, 중국, EU |
| **T2** | 🔵 파랑 | 커넥터 | 일본, 영국, 한국 |
| **T3** | 🟢 초록 | 액티브 | 주요 기업, 기관 |
| **T4** | ⚫ 회색 | 일반 | 중소 기업 |
| **Ghost** | ⬛ 진회색 | 비활성 | 휴면 계정 |

### 2.2 노드 클릭

노드를 클릭하면 상세 정보 패널이 열립니다:

- **이름/타이틀**: 노드 식별 정보
- **Tier**: 노드 등급
- **RV (Real Value)**: 실제 가치
- **NA (Net Assets)**: 순자산
- **섹터/지역**: 분류 정보

### 2.3 모션 (Motion)

노드 간의 연결선이 **모션**입니다. 자금, 무역, 투자 등의 흐름을 나타냅니다.

- 선 굵기: 금액 규모
- 점선: 진행 중인 거래
- 색상: 거래 유형

### 2.4 필터링

상단 필터 버튼으로 특정 데이터만 볼 수 있습니다:

- **Tier 필터**: T1, T2, T3, T4 선택
- **섹터 필터**: Tech, Finance, Energy 등
- **지역 필터**: Americas, Europe, Asia 등

### 2.5 검색

검색창에 이름이나 ID를 입력하면 해당 노드로 이동합니다.

## 3. 데이터 소스 연동

### 3.1 지원 데이터 소스

AUTUS는 12개의 글로벌 데이터 소스를 지원합니다:

#### 무역/물류
- **UN Comtrade**: 국가간 무역 통계
- **UNIPASS**: 한국 관세청 통관 데이터
- **FAO**: 농산물/식량 무역

#### 금융/투자
- **World Bank**: FDI, GDP 데이터
- **SEC 13F/Form4**: 기관투자자 보유 현황

#### 에너지
- **EIA**: 원유, LNG, 석탄 무역

#### 기타
- **GitHub**: 기술 협력 네트워크
- **UNHCR/IOM**: 난민/이주 데이터
- **SIPRI**: 방산/무기 무역

### 3.2 데이터 새로고침

데이터는 자동으로 업데이트됩니다. 수동 새로고침:

```
API: POST /api/motion/pipeline/run
```

## 4. 대시보드 활용

### 4.1 주요 지표

- **총 노드 수**: 모니터링 중인 엔티티 수
- **모션 수**: 활성 거래/흐름 수
- **총 가치**: 전체 RV 합계

### 4.2 줌 레벨 가이드

| 줌 레벨 | 표시 범위 | 용도 |
|---------|----------|------|
| 2-3 | 전체 세계 | 글로벌 트렌드 파악 |
| 4-5 | 대륙/지역 | 지역별 비교 |
| 6-7 | 국가 | 국가 내 상세 |
| 8+ | 도시/기업 | 개별 엔티티 분석 |

## 6. 자주 묻는 질문

### Q: 데이터는 얼마나 자주 업데이트되나요?

**A:** 대부분의 데이터 소스는 일 단위로 업데이트됩니다. 실시간 데이터는 WebSocket을 통해 제공됩니다.

### Q: 특정 노드가 보이지 않아요

**A:** 줌 레벨을 확인하세요. T4 노드는 줌 레벨 6 이상에서만 표시됩니다. 필터 설정도 확인해주세요.

### Q: 데이터를 내보낼 수 있나요?

**A:** API를 통해 JSON 형태로 데이터를 가져올 수 있습니다:
```
GET /api/motion/realtime
```

### Q: 자체 데이터를 추가할 수 있나요?

**A:** 관리자 권한으로 커스텀 노드와 모션을 추가할 수 있습니다. 상세 내용은 개발자 문서를 참조하세요.

### Q: 모바일에서도 사용 가능한가요?

**A:** 기본적인 조회는 가능하지만, 최적의 경험을 위해 데스크톱 브라우저를 권장합니다.
