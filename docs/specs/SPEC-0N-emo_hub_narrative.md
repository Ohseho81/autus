# SPEC-0N: EMO Hub Narrative

## 한 줄 정의
EMO Hub는 시설(건물/설비)의 유지보수·운영을 자동화하는 시설관리 OS이다.

## 이해관계자
| 역할 | 설명 |
|------|------|
| 시설관리자 | 전체 시설 현황 모니터링, 의사결정 |
| 현장기사 | 점검/수리 업무 수행 |
| 입주자 | 이슈 신고, 서비스 요청 |
| 외부업체 | 전문 수리/점검 |

## 하루 타임라인
| 시간 | 이벤트 |
|------|--------|
| 05:00 | 시스템 자동 점검 시작 |
| 07:00 | 현장기사 출근, 오늘 작업 할당 |
| 09:00 | 입주자 이슈 신고 시작 |
| 12:00 | 긴급 수리 완료 현황 점검 |
| 18:00 | 일일 리포트 자동 생성 |
| 22:00 | 야간 모니터링 모드 전환 |

## 대표 사건 3개

### 1. 보일러 고장
- 센서가 이상 감지 → 티켓 자동 생성
- 기사 할당 → 현장 출동 → 수리 완료
- 결과 기록 → 예방정비 스케줄 업데이트

### 2. 위생 점검
- 월간 점검 스케줄 자동 생성
- 체크리스트 배포 → 점검 수행 → 결과 제출
- 미비사항 티켓 생성 → 조치 완료

### 3. 화재 훈련
- 연간 훈련 일정 등록
- 사전 공지 → 훈련 실시 → 결과 기록
- 개선사항 도출 → 다음 훈련에 반영
