| 퇴원 사유 파악 어려움 | "학부모가 전화 안 받음", "사유 안 밝힘" | AI 분석 → 퇴원 패턴 예측 |

## 2. 퇴원 위험 신호 (AUTUS 감지 대상)

| 신호 | 감지 방법 | 위험도 | 점수 |
|------|----------|--------|------|
| 출석률 급락 | 출결 데이터 | 🔴 Critical | +40 |
| 숙제 미제출 증가 | 과제 제출률 | 🔴 Critical | +35 |
| 성적 하락 | 시험 점수 추이 | 🟠 High | +25 |
| 학부모 연락 두절 | 상담 응답률 | 🟠 High | +25 |
| 수업 태도 변화 | 강사 피드백 | 🟡 Medium | +15 |
| 미납 발생 | 수납 데이터 | 🟡 Medium | +15 |

**위험도 계산**: `churn_risk_score = SUM(detected_signals)`
- 80+ → 🔴 Critical (즉시 상담)
- 50-79 → 🟠 High (1주 내 상담)
- 30-49 → 🟡 Medium (모니터링)

## 3. Outcome 기반 과금 구조

### 3.1 핵심 Outcome 정의

| Outcome | 측정 방법 | 가치 산정 |
|---------|----------|----------|
| **퇴원 방지** | 상담 후 retention | 월 수업료 × 6개월 × 10% |
| **미수금 회수** | 결제 완료 이벤트 | 회수 금액 × 10% |
| **신규 등록 전환** | 상담 → 등록 | 첫 달 수업료 × 10% |

### 3.2 가치 계산 예시

```
퇴원 방지 1건 가치:
= 월 수업료(35만원) × 잔여 개월(6개월) × 기여율(10%)
= 35만원 × 6 × 0.1
= 21만원
```

### 3.3 과금 티어

| 티어 | 월 Outcome 가치 | 과금 비율 | 예상 비용 |
|------|----------------|----------|----------|
| **Starter** | ~100만원 | **무료** | 0원 |
| **Growth** | 100~500만원 | 5% | 5~25만원 |
| **Scale** | 500~2000만원 | 7% | 35~140만원 |
| **Enterprise** | 2000만원+ | 10% | 200만원+ |

```
┌─────────────────────────────────────────────────┐
│  1순위: 학원노 (다음카페) - 137,000+ 회원        │
│  2순위: 훈장마을 - 학원 DB 보유                  │
│  3순위: 지역 맘카페 - 학부모 입소문              │
│  4순위: 지인 네트워크 - 신뢰 기반                │
└─────────────────────────────────────────────────┘
```

### 4.1 학원노 접근 전략

```
1단계: 가입 (카카오톡 연동)
  ↓
2단계: 1주일 모니터링 (퇴원/미수금 글 수집)
  ↓
3단계: 가치 제공 글 작성
  - "퇴원 위험 학생 조기 발견하는 3가지 신호"
  ↓
4단계: 파일럿 모집 글
  - 3개월 무료, 효과 없으면 0원
  ↓
5단계: 개별 DM
```

## 5. 파일럿 모집 메시지

### 5.1 게시글 템플릿

```
[제목] 퇴원 위험 학생, 미리 알 수 있다면?

안녕하세요, 원장님들.

저희 팀은 학원 운영 데이터를 분석해서
퇴원 위험이 높은 학생을 미리 찾아주는 시스템을 개발하고 있습니다.

출석률, 숙제 제출, 성적 변화, 상담 이력을 종합 분석해서
"이 학생, 다음 달 퇴원 확률 80%"
이런 식으로 알려드리고,

학부모 상담할 때 쓸 수 있는 스크립트까지 AI가 생성해줍니다.

------

[파일럿 프로그램]
- 기간: 3개월 무료
- 대상: 학생 50~200명 규모 학원 3~5곳
- 조건: 주 1회 피드백, 퇴원/미수금 데이터 공유

[과금 방식]
- 실제로 퇴원 방지된 건수 × 가치의 5~10%
- 효과 없으면 0원

------

관심 있으신 원장님, 댓글이나 쪽지 주세요.
```

### 5.2 개별 DM 템플릿

```
원장님 안녕하세요,

학원노 카페에서 퇴원 관련 글 남기신 거 보고 연락드립니다.

저희가 AI로 퇴원 위험 학생을 미리 찾아주는 시스템을 만들고 있는데,
파일럿 학원을 모집하고 있어요.

- 3개월 무료
- 효과 없으면 과금 0원
- 효과 있으면 퇴원 방지 가치의 10%만

혹시 10분 정도 통화 가능하실까요?
```

## 8. 핵심 메시지

> **"퇴원 막으면 돈 벌고, 못 막으면 무료."**

이 메시지가 통하는 이유:
1. **리스크 제로** - 효과 없으면 0원
2. **명확한 가치** - 퇴원 1건 = 210만원 (35만원 × 6개월)
3. **핵심 페인포인트 직격** - 퇴원 = 원장의 가장 큰 스트레스
