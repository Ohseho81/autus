## 🟢 Phase 3: 도메인 확장 (2-3개월)

### 3.1 학원 도메인 완성
```
□ 학원 템플릿 상세화
□ 학원 벤치마크 데이터 수집
□ 학원 특화 리포트
□ 학원장 온보딩 플로우
□ 학원 데모 데이터 10개
```

### 3.2 새 도메인 추가
```
□ 식당 (RESTAURANT) 템플릿
   - 노드: 테이블회전율, 객단가, 재료비율, 리뷰점수
   - 임계값: 객단가 하락, 리뷰 악화
   
□ 프리랜서 (FREELANCER) 템플릿
   - 노드: 프로젝트수, 시간당단가, 반복고객율
   - 임계값: 파이프라인 고갈, 번아웃 지표
   
□ 소매점 (RETAIL) 템플릿
   - 노드: 재고회전, 마진율, 방문객수
   - 임계값: 재고 적체, 마진 하락
   
□ 콘텐츠 크리에이터 (CREATOR) 템플릿
   - 노드: 구독자, 조회수, 광고단가, 스폰서십
   - 임계값: 알고리즘 변화, 번아웃
```

### 3.3 멀티 개체 지원
```
□ 하나의 계정에 여러 개체 관리
□ 개체 간 비교 분석
□ 그룹 통계 (예: 5개 학원 통합)
□ 벤치마크 비교
```
