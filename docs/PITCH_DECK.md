## Slide 1: 문제

### 📉 학원 원장의 가장 큰 고민

```
"왜 갑자기 그만두는 거죠?"

학원 평균 연간 이탈률: 30-40%
이탈 1명 = 월 30만원 × 12개월 = 연 360만원 손실
100명 학원 기준: 연간 1억+ 손실

그런데... 이탈 징후를 미리 알 수 있다면?
```

**핵심 인사이트:**
- 학부모의 70%는 "갑자기" 그만둔다고 느낌
- 실제로는 평균 2-3주 전부터 징후 존재
- 하지만 선생님들은 너무 바빠서 놓침

## Slide 2: 솔루션

### 🌡️ AUTUS = 학생 "온도" 측정기

```
A = T^σ
(Activity = Temperature × Sigma)

30초 기록 → AI 이탈 예측 → 사전 대응
```

**작동 방식:**
1. 선생님이 수업 후 30초 Quick Tag 입력
2. AI가 학생 "온도" 계산 (0-100°)
3. 온도 낮은 학생 자동 알림
4. 선제적 케어 → 이탈 방지

**결과:**
- 이탈률 30% 감소
- 월 매출 5-10% 증가
- 선생님 업무 부담 감소

```
┌─────────────────────────────────────────────────────────────┐
│  σ (Sigma) = 학생 상태 종합 지수                            │
│                                                             │
```
