# ⚛️ AUTUS Physics Type System

## 핵심 개념: F = ma

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   F = ma 에서 m이 없으면 운동 방정식이 불완전하다                           │
│                                                                             │
│   같은 충격(F)이라도:                                                       │
│   - 스타트업: 질량 작음 → 가속도 큼 (빠르게 변함)                          │
│   - 대기업: 질량 큼 → 가속도 작음 (천천히 변함)                            │
│   - 국가: 질량 거대 → 거의 안 변함 (관성)                                  │
│                                                                             │
│   타입 = 운동 방정식의 계수 (질량/관성)                                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 📐 타입별 특성

| 타입 | 관성 | K변화율/일 | 임계점 | 수명 | 핵심 슬롯 |
|------|------|-----------|--------|------|-----------|
| INDIVIDUAL (개인) | 0.10 | ±5%/일 | K<-0.5 | 80년 | BOND, MENTOR |
| STARTUP (스타트업) | 0.20 | ±10%/일 | K<-0.3 | 3-5년 | SUPPLIER, CLIENT |
| SMB (중소기업) | 0.40 | ±3%/일 | K<-0.4 | 10-20년 | CLIENT, PEER |
| ENTERPRISE (대기업) | 0.80 | ±1%/일 | K<-0.6 | 50년+ | ALLY, RIVAL |
| CITY (도시) | 0.90 | ±0.5%/일 | K<-0.7 | 100년+ | ORIGIN, ALLY |
| NATION (국가) | 0.95 | ±0.1%/일 | K<-0.8 | 200년+ | ALLY, ADVERSARY |
| IDEOLOGY (이념) | 0.99 | ±0.01%/일 | K<-0.9 | 1000년+ | DISCIPLE, ADVERSARY |

### 관성 설명
- **0**: 즉시 반응 (불안정)
- **0.5**: 중간 (균형)
- **1**: 거의 안 변함 (안정적)

## ⚡ 운동 방정식

### 기본 방정식
```
가속도 = F / (1 + 관성)
ΔK = α × 가속도 × 생명력 × (1 - |K|)
```

### 예측 방정식
```python
# 현재 예측 (타입 없음)
K(t+1) = K + dK/dt × t

# 타입 적용 예측
K(t+1) = K + (dK/dt × t) / (1 + 관성)
```

## 💻 사용법

### 기본 사용

```python
from backend.physics import (
    TypedPhysicsEngine, 
    EntityType,
    print_type_dashboard
)

# 엔진 생성
engine = TypedPhysicsEngine(alpha=0.1)

# 엔티티 생성
engine.create_entity("startup_a", EntityType.STARTUP, "테크스타트업")
engine.create_entity("samsung", EntityType.ENTERPRISE, "삼성전자")

# 힘 적용
result = engine.apply_force("startup_a", force=1.0, context="투자 유치")
print(f"ΔK: {result['delta_k']}")  # 빠른 변화

result = engine.apply_force("samsung", force=1.0, context="시장 변화")
print(f"ΔK: {result['delta_k']}")  # 느린 변화

# 상호작용 계산
interaction = engine.calculate_interaction("samsung", "startup_a", raw_effect=1.0)
print(f"효과: {interaction['final_effect']}")  # 1.8x 증폭

# 궤적 예측
trajectory = engine.predict_trajectory("startup_a", days=30)

# 대시보드
print_type_dashboard(engine)
```
