# 🏛️ AUTUS v1.0 — Final Consolidation

> **이 문서는 의사결정 기준 문서로 그대로 고정 가능하다.**

---

## 1. AUTUS의 정체성 (고정)

- AUTUS는 **툴이 아니다**
- AUTUS는 **설명하는 시스템이 아니다**
- AUTUS는 **결정을 도와주지 않는다**
- AUTUS는 **환경처럼 작동하는 의사결정 게이트**다

> 사람을 바꾸지 않고
> **사람이 움직일 수 있는 범위만 바꾼다**

| 페이지 | 구성 |

- ❌ KPI/점수

## 5. 자동 전환 조건표 (Hard Rules)

> **"설명 소멸은 선택이 아니라 조건이다"**
> 사람이 판단하지 않는다. AUTUS가 관측값만으로 자동 실행한다.

### 전환 변수 (Core Metrics)

| 변수 | 의미 | 측정 방식 |
|------|------|----------|
| **T** | 사용 기간 | 최초 실행 기준 일수 |
| **Gᵣ** | Gate 체감 횟수 | RING/LOCK 발생 수 |
| **Rᵣ** | 재시도율 | 동일 행동 반복 비율 |
| **ΔṠ** | 엔트로피 변화율 | 행동-환경 마찰 |
| **Iₛ** | 행동 안정성 | 실행 패턴 분산 |

### K2 (일반 사용자)

| 현재 Phase | 전환 조건 | 다음 Phase |
|------------|----------|------------|
| Phase 2 | T ≥ 30일 AND Gᵣ ≥ 3 | Phase 2.5 |
| Phase 2.5 | T ≥ 90일 AND Rᵣ ↓ 40% | Phase 3 (부분) |
| Phase 3 (부분) | Iₛ ≥ 기준값 AND ΔṠ 안정 | Phase 3 (전면) |

### K4–K6 (운영/설계)

| 조건 | 적용 Phase |
|------|-----------|
| 최초 진입 | Phase 2.5 |
| Gᵣ ≥ 1 OR Afterimage 열람 | Phase 3 |

### K10 (관측자)

| 조건 | 적용 Phase |
|------|-----------|
| 최초 진입 | Phase 3 |
| Afterimage ≥ N회 재생 | Phase 4 |

### 전환 시 절대 하지 않는 것

- ❌ 알림
- ❌ 공지
- ❌ "업데이트 안내"
- ❌ 왜 바뀌었는지 설명

> 사용자는 "아, 요즘 이런 식이네" 정도로만 인식해야 한다.
