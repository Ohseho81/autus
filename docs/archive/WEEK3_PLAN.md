## 구현할 것

### Week 3: Data Engine
```
core/data/
├── __init__.py
├── collector.py       # 데이터 수집
├── storage.py        # 로컬 저장소
└── analyzer.py       # 데이터 분석
```

**목표**:
- 사용자 행동 패턴 수집
- 로컬 저장 (Privacy 보장)
- 기본 분석

### Week 4: Wisdom Engine
```
core/wisdom/
├── __init__.py
├── extractor.py      # 지혜 추출
├── ranker.py        # 패턴 순위화
└── recommender.py   # 추천 시스템
```

**목표**:
- 효율적인 패턴 발견
- 베스트 프랙티스 추출
- 자동 추천
