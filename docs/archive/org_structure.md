# Autus Organization Structure (Zero-Action Model)

## 1) Kernel Team (1명)
- Autus Kernel Architect
- Only maintains the kernel laws.
- No modules, no features, no operations.

## 2) Module Contributors (∞명)
- Pack creators
- Policy creators
- Connector creators
- UI builders
- Workflow designers

## 3) Cell Owners
- Cities, schools, enterprises
- Submit requirements
- Install Packs
- Operate Packs
- Create patterns

## 4) Ecosystem Council (Optional)
- Dispute resolution
- Version stabilization
