## 타임라인

### 0:00-0:03 — INTRO (3초)

**화면**: 검은 화면 → SOLAR 페이드인

```
┌──────────────────────────────────┐
│                                  │
│         ○   ○   ☉   ○           │
│                                  │
│                                  │

│  15%                         ●   │
```

- 부드러운 궤도 회전
- 녹색 발광
- 안정된 상태
