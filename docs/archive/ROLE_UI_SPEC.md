# AUTUS Role-Based UI Specification (LOCK)

> **"같은 현실, 다른 시선"**
> 모든 역할은 동일한 시스템 상태를 보지만, 각자의 책임에 맞는 손실만 본다.

---

## 📋 5개 역할 정의

| Role | 한글 | 대상 | 핵심 관심사 |
|------|------|------|-------------|
| **SUBJECT** | 주체 | 학생, 구직자, 피고용인 | 내 생존 시간 |
| **OPERATOR** | 운영자 | PM, 매니저, 코디네이터 | 전체 인원 리스크 |
| **SPONSOR** | 후원자 | 투자자, 정부 지원기관 | 투자 효율 (ROI) |
| **EMPLOYER** | 고용주 | 기업, HR 담당자 | 인력 확보/이탈 |
| **INSTITUTION** | 기관 | 대학, 정부, 협회 | 시스템 거버넌스 |

---

## 🎨 공통 UI 원칙 (모든 Role 적용)

### 불변 규칙

```
1. 버튼: 최대 1개
2. 설명 텍스트: 0
3. 뒤로가기: 불가
4. 상세 페이지: 없음
5. 툴팁/호버: 없음
6. 애니메이션: 상태 표현만 (장식 ❌)
```

### 공통 레이아웃

```
┌─────────────────────────────────────────────┐
│  [ROLE BADGE]              [GATE] [STATUS]  │  ← Header (고정)
├─────────────────────────────────────────────┤
│                                             │
│              LOSS DISPLAY                   │  ← 핵심 손실 표시
│              (역할별 다름)                   │
│                                             │
├─────────────────────────────────────────────┤
│                                             │
│              SOLAR SYSTEM                   │  ← 9 Planets (공통)
│              (상태 시각화)                   │
│                                             │
├─────────────────────────────────────────────┤
│              [ACTION BUTTON]                │  ← CTA (조건부)
│              [COUNTDOWN]                    │
└─────────────────────────────────────────────┘
```

## 1️⃣ SUBJECT View (주체)

### 대상
- 필리핀 석사 학생
- 취업 준비생
- 현재 고용 상태인 개인

### 핵심 지표

| 지표 | 표시 형식 | 의미 |
|------|----------|------|
| **SURVIVAL** | `216일` | 현재 상태로 버틸 수 있는 기간 |
| **BURN** | `−₩47만/월` | 월간 순 지출 |
| **RISK** | `32%` | 위험도 |

### UI 특징

```
┌─────────────────────────────────────────────┐
│  👤 SUBJECT                    GREEN   OK   │
├─────────────────────────────────────────────┤
│                                             │
│              SURVIVAL                       │
│               216일                         │
│            ━━━━━━━━━━━━━                    │
│                                             │
│         BURN: −₩47만/월                     │
│                                             │
├─────────────────────────────────────────────┤
│                                             │
│              ☀️ SOLAR                       │
│                                             │
├─────────────────────────────────────────────┤
│                                             │
│              💰 −32%                        │
│         ████ RECOVER ████                   │
│                  5                          │
└─────────────────────────────────────────────┘
```

### 색상
- 메인: `#00ff88` (Green)
- 배경: `#000000`
- 위험: `#ff4444`

### ACTION 조건
- `risk >= 40%` AND `gate !== 'RED'` → 버튼 노출
- `survival < 90일` → 강조 표시

- `at_risk >= 1` → 버튼 노출
- `critical >= 1` → 즉시 실행 강조

## 📊 Role별 ACTION 매핑

| Role | ACTION 이름 | 조건 | 효과 |
|------|-------------|------|------|
| SUBJECT | RECOVER | risk >= 40% | 개인 리스크 감소 |
| OPERATOR | INTERVENE | at_risk >= 1 | 위험 인원 개입 |
| SPONSOR | OPTIMIZE | efficiency < 80% | 투자 최적화 |
| EMPLOYER | RETAIN | churn_risk >= 1 | 이탈 방지 |
| INSTITUTION | *(없음)* | - | 모니터링만 |
