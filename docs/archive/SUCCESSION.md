# 4. Level 2: 승계자 3인

## 4.1 자격
```yaml
승계자_자격:
  필수:
    - AUTUS 기여 1년 이상
    - 헌법 완전 이해
    - 커뮤니티 신뢰
    - 기술적 역량
  
  금지:
    - 경쟁 프로젝트 관여
    - 헌법 위반 이력
    - 이해충돌
```

## 4.2 선정 방법
```
1. Seho가 생전 지명 (비공개 봉인)
2. 지명 없을 시: 커뮤니티 투표
3. 후보: 상위 기여자 10인
4. 선출: 득표 상위 3인
```

## 4.3 권한
```yaml
승계자_권한:
  가능:
    - Oracle 접근 (3인 중 2인 동의)
    - 방향 제시
    - 헌법 해석 (위원회와 협의)
    - 긴급 결정 (2/3 동의)
  
  불가:
    - 헌법 수정 (제1-6, 10-13조)
    - 단독 결정
    - Oracle 데이터 외부 공개
```

## 4.4 현재 지명자
```
# 이 섹션은 Seho가 업데이트

승계자_1: [미지명]
승계자_2: [미지명]
승계자_3: [미지명]

봉인_위치: [비공개]
봉인_해제_조건: Level 2 트리거 시
```
