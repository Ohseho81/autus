## 📋 카테고리 총괄

| 카테고리 | 컴포넌트 수 | 용도 |
|---------|-----------|------|
| Header Menu | 12+ | 네비게이션, 상단바 |
| Media (Images/Videos/Sliders) | 15+ | 미디어 표시 |
| E-Shop | 18+ | 이커머스, 상품 |
| Register/Login | 8+ | 인증, 회원가입 |
| Blog/News/Magazine | 12+ | 콘텐츠 피드 |
| Post/Article | 10+ | 게시글 상세 |
| Portfolio (Mixed) | 15+ | 포트폴리오 |
| Portfolio (Images) | 12+ | 이미지 갤러리 |

## 1️⃣ HEADER MENU 컴포넌트

### 기본 헤더 유형
```
┌─────────────────────────────────────────────────┐
│ [Logo]  Nav1  Nav2  Nav3  Nav4     [Search] [☰] │
└─────────────────────────────────────────────────┘
```

### 변형
| ID | 유형 | 특징 | AUTUS 적용 |
|----|------|------|-----------|
| H1 | 기본 헤더 | 로고 + 네비게이션 | ✅ TopBar |
| H2 | 검색 포함 | 검색창 통합 | Command 패널 |
| H3 | CTA 버튼 | 우측 액션 버튼 | Gate 버튼 |
| H4 | 드롭다운 | 서브메뉴 지원 | Layer 선택 |
| H5 | 메가 메뉴 | 전체화면 드롭다운 | Dashboard 네비 |
| H6 | 투명 헤더 | 배경 오버레이 | Solar HQ |
| H7 | 고정 헤더 | 스크롤 시 고정 | 모든 페이지 |
| H8 | 햄버거 | 모바일 메뉴 | 반응형 |

### CSS 변수
```css
--header-height: 64px;
--header-bg: rgba(10, 10, 15, 0.95);
--header-border: 1px solid rgba(255,255,255,0.1);
```

## 2️⃣ MEDIA 컴포넌트 (Images/Videos/Sliders)

```
┌─────────────────┐   ┌────┬────┬────┐   ┌─────┬─────┐
│                 │   │    │    │    │   │     │     │
│   Hero Image    │   │ 1  │ 2  │ 3  │   │  A  │  B  │
│                 │   │    │    │    │   │     ├─────┤
│                 │   ├────┴────┴────┤   │     │  C  │
└─────────────────┘   │   Caption    │   └─────┴─────┘
     단일 히어로           3열 그리드        비대칭 그리드
```

### 슬라이더 유형
| ID | 유형 | 특징 |

| S5 | 전체화면 | 풀스크린 갤러리 |

### 비디오 유형
| ID | 유형 | 특징 |
|----|------|------|
| V1 | 인라인 | 콘텐츠 내 삽입 |
| V2 | 모달 | 클릭 시 모달 재생 |
| V3 | 배경 | 무음 자동재생 배경 |
| V4 | 썸네일 | 재생 버튼 오버레이 |

## 3️⃣ E-SHOP 컴포넌트

### 상품 카드
```
┌─────────────────┐
│   [Product      │
│    Image]       │
│                 │
├─────────────────┤
│ Product Name    │
│ ★★★★☆ (42)     │
│ $99.00  $129    │
│ [Add to Cart]   │
└─────────────────┘
```

### 상품 목록 레이아웃
| ID | 유형 | 열 수 | 용도 |
|----|------|-------|------|
| P1 | 그리드 | 3-4열 | 카테고리 페이지 |
| P2 | 리스트 | 1열 | 상세 비교 |
| P3 | 매거진 | 혼합 | 프로모션 |
| P4 | 캐러셀 | 가변 | 추천 상품 |

### 장바구니/결제
```
Cart → Address → Payment → Confirmation
 [1]     [2]       [3]        [4]
```

| ID | 컴포넌트 | 요소 |
|----|---------|------|
| C1 | 장바구니 | 상품 목록, 수량, 소계 |
| C2 | 배송 정보 | 주소 입력 폼 |
| C3 | 결제 | 카드/간편결제 선택 |
| C4 | 확인 | 주문 요약, 확인 버튼 |
| C5 | 위시리스트 | 저장된 상품 |

```
│                         │
```

```
┌─────────────────────────────────────────────────┐
│ Category                                        │

│ Article Title                                   │

│                                                 │
│ [Avatar] Author Name · Jan 18, 2026 · 8 min     │
│ [Share] [Save] [...]                            │
├─────────────────────────────────────────────────┤
│                                                 │
│ [Hero Image]                                    │
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│ Article content starts here with a drop cap    │
│ letter. The rest of the paragraph continues... │
│                                                 │
│ ## Subheading                                   │
│                                                 │
│ More content with embedded images, quotes,     │
│ and other rich media elements.                 │
│                                                 │
│ > "Blockquote styling for important quotes"    │
│                                                 │

│ Tags: [tag1] [tag2] [tag3]                      │
├─────────────────────────────────────────────────┤
│ Related Articles                                │
│ ┌────┐ ┌────┐ ┌────┐                           │
```

## 🔧 AUTUS 적용 매핑

### Solar HQ 페이지
| 섹션 | 참조 컴포넌트 | 구현 |
|------|-------------|------|
| TopBar | H6 (투명 헤더) | ✅ |
| 3D Globe | V3 (배경 비디오) | Three.js |
| Info Panel | T5 (콜아웃) | 좌측 패널 |
| Action Panel | C3 (버튼 그룹) | 우측 패널 |
| Audit Modal | A5 (확인 폼) | 모달 |

### Dashboard 페이지
| 섹션 | 참조 컴포넌트 | 구현 |
|------|-------------|------|
| Header | H1 + H4 | 네비게이션 |
| Stats | P1 (그리드) | 카드 그리드 |
| Charts | B4 (카드 그리드) | 차트 영역 |
| Activity | B3 (타임라인) | 로그 |

### Command 페이지
| 섹션 | 참조 컴포넌트 | 구현 |
|------|-------------|------|
| Search | H2 (검색 헤더) | 검색바 |
| Results | B5 (리스트) | 명령어 목록 |
| Actions | C1 (장바구니) | 액션 큐 |

## 📐 반응형 브레이크포인트

```css
/* Mobile First */
--bp-sm: 640px;   /* 모바일 */
--bp-md: 768px;   /* 태블릿 */
--bp-lg: 1024px;  /* 데스크톱 */
--bp-xl: 1280px;  /* 와이드 */
--bp-2xl: 1536px; /* 울트라와이드 */
```

### 그리드 열 수
| 브레이크포인트 | 카드 그리드 | 갤러리 |
|--------------|-----------|--------|
| Mobile | 1열 | 2열 |
| Tablet | 2열 | 3열 |
| Desktop | 3열 | 4열 |
| Wide | 4열 | 5열 |
