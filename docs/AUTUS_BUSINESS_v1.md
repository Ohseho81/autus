SOM (Serviceable Obtainable Market)
═══════════════════════════════════════════════════

- 타겟 학원: 2,000개 (80,000개 중 2.5%)

## B.3 경쟁 해자 (Moat)

### 4가지 해자

1. **개념 해자 (Conceptual Moat)**
   - "시간 기반 관계 가치 측정"이라는 새로운 카테고리 정의
   - CRM: 거래 중심 (시간 개념 없음)
   - HR: 내부만 (고객 관계 없음)
   - META: 연결만 (가치 측정 불가)
   - → AUTUS = 유일한 "시간 기반" 솔루션

2. **데이터 해자 (Data Moat)**
   - λ, σ 축적 → 예측 정확도 향상
   - 1년 사용 → σ 예측 70% 정확도
   - 3년 사용 → σ 예측 90% 정확도
   - → 후발 주자는 데이터 없이 시작해야 함

3. **네트워크 해자 (Network Moat)**
   - 조직 간 벤치마크 → 산업 표준화
   - "우리 학원 λ는 산업 평균 대비 어떤가?"
   - → 참여자 증가 = 데이터 가치 증가

4. **전환 비용 해자 (Switching Cost Moat)**
   - 축적된 λ, σ, V 히스토리
   - 3년간 축적된 관계 가치 데이터
   - → 타 솔루션 이동 시 처음부터 시작

## B.4 실증 케이스: 100명 학원 시뮬레이션

### 시나리오 설정

```
가상 학원 "스마트 학원"

규모:
- 학생: 100명
- 직원: 원장 1, 실장 1, 강사 5, 행정 1 = 8명
- 학부모: 100명
- 총 노드: 208개

재무:
- 월 매출: ₩45,000,000
- 평균 수업료: ₩450,000/월
- 이탈률: 8%/월 (Before AUTUS)
```

### λ 설정

| 역할 | 수 | λ | 총 λ |
|------|-----|-----|-------|
| 원장 | 1 | 5.0 | 5.0 |
| 실장 | 1 | 3.5 | 3.5 |
| 강사 | 5 | 2.0 | 10.0 |
| 행정 | 1 | 1.5 | 1.5 |
| 학생 | 100 | 1.0 | 100.0 |
| 학부모 | 100 | 1.2 | 120.0 |
| **합계** | **208** | | **240.0** |

평균 λ = 1.15

### ROI 계산

```
AUTUS 도입 ROI (6개월)

비용:
├─ AUTUS 구독료: ₩299,000 × 6 = ₩1,794,000
└─ 세팅/교육: ₩500,000
총 비용: ₩2,294,000

효과:
├─ 이탈률 감소: 8% → 5%
├─ 이탈 방지: 3명 × ₩450,000 = ₩1,350,000/월
└─ 6개월 총 효과: ₩8,100,000

ROI = (₩8,100,000 - ₩2,294,000) / ₩2,294,000 = 253%
회수 기간: 1.7개월
```
