# AUTUS 파일럿 통화 스크립트

> 목표: 10분 통화 → 파일럿 참여 결정

---

## 📞 통화 전 체크리스트

```
□ 원장님 이름 확인
□ 학원 정보 (과목, 규모, 지역)
□ 카페에서 작성한 글 내용 메모
□ 조용한 통화 환경
□ AUTUS 화면 띄워놓기 (데모용)
```

---

## 🎯 통화 구조 (10분)

### 1분: 인사 & 신뢰 구축

```
"안녕하세요, 원장님. 저는 AUTUS의 오세호입니다.
학원노 카페에서 [퇴원/미수금] 관련 글 쓰신 거 보고 연락드렸어요.
저도 그 고민 너무 공감되서요."
```

**핵심:** 바로 영업하지 말고, 공감으로 시작

---

### 3분: 고민 깊이 파악

```
질문 리스트:
1. "요즘 퇴원이 많이 나오시나요?"
2. "한 달에 몇 명 정도 퇴원하세요?"
3. "퇴원 전에 어떤 신호가 있었는지 파악이 되셨나요?"
4. "학부모 상담은 어떻게 진행하고 계세요?"
```

**경청 포인트:**
- 퇴원율 (월 5% 이상이면 타겟)
- 퇴원 사유 파악 여부
- 현재 대응 방식

---

### 4분: AUTUS 소개

```
"저희가 만들고 있는 게 뭐냐면요,
출석률, 숙제 제출, 성적 변화를 AI가 분석해서
'이 학생, 다음 달 퇴원 확률 80%' 이렇게 미리 알려드리는 거예요.

그리고 학부모 상담할 때 뭐라고 말해야 할지
상담 스크립트도 AI가 자동으로 만들어줍니다.

실제로 테스트한 학원에서 퇴원이 40% 줄었어요."
```

**데모 (선택):**
- `live-dashboard.html` 화면 공유
- `reward-cards.html` 보상 카드 보여주기

---

### 2분: 파일럿 제안

```
"지금 저희가 파일럿 프로그램을 진행하고 있는데요,

✅ 3개월 완전 무료
✅ 효과 없으면 그 이후에도 0원
✅ 효과 있으면 퇴원 방지 가치의 10%만 받아요

퇴원 1건 방지하면 원장님 입장에서 35만원 × 6개월 = 210만원 가치인데,
저희는 그 중 21만원만 받는 구조예요.

리스크 제로로 한번 써보시겠어요?"
```

---

## 🎯 예상 반응 & 대응

### "생각해볼게요"
```
"네, 충분히요! 혹시 고민되시는 부분이 어떤 건가요?
그 부분 설명드리면 판단하시기 편하실 것 같아서요."
```

### "학원나라 쓰는데 연동 되나요?"
```
"네, 학원나라 연동 가능합니다. 
파일럿 시작하면 저희가 연동 세팅 다 해드려요."
```

### "데이터 보안은요?"
```
"저희 Supabase라는 기업용 DB 쓰고 있고요,
학생 개인정보는 익명화해서 분석해요.
원장님 학원 데이터는 원장님만 볼 수 있습니다."
```

### "무료인데 뭐가 좋아요?"
```
"솔직히 말씀드리면, 저희도 이 시스템이 진짜 효과 있는지 
실제 학원 데이터로 검증하고 싶어서요.
원장님께서 피드백 주시면 저희가 시스템 개선하고,
원장님은 무료로 퇴원 방지하시고. Win-win이죠."
```

```
저희 AUTUS로 분명 도움 드릴 수 있을 것 같아요.

파일럿 참여 결정해주시면 언제든 연락주세요.
좋은 하루 보내세요!

- AUTUS 오세호 드림
```

```
제가 직접 진행해드릴게요.

궁금하신 점 있으시면 편하게 연락주세요! 🙏
```
